# Real-Time API Monitoring & Performance Analytics System

A real-time monitoring dashboard that tracks API performance, response latency, request volume, and system logs using event-driven architecture.

---

## Features

- Real-time API monitoring
- Kafka-based event streaming
- API latency tracking
- Error rate monitoring
- Task management system
- Audit logs for API activity
- Performance visualization dashboard

---

## Tech Stack

Frontend

- React
- Chart.js
- Tailwind CSS

Backend

- Node.js
- Express.js

Database

- PostgreSQL (Neon)

Messaging

- Apache Kafka

Deployment

- Render
- Vercel
